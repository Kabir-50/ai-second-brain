# AI Second Brain – Personal Knowledge Assistant

A production-level RAG-based document chat system.

## Stack
- Frontend: React + TailwindCSS
- Backend: Node.js + Express
- AI: Ollama (Llama3/Mistral) + LangChain
- Vector DB: ChromaDB
- Database: MongoDB
- DevOps: Docker + GitHub Actions

## Structure
/frontend      → React app
/backend       → Express API
/ai-services   → RAG pipeline (Python/Node)
/docker        → Dockerfiles & compose
/docs          → Architecture docs
